## 🚀 Overview

Minimastic is a modern AI-powered quiz platform designed to make learning engaging, interactive, and accessible.

Users can choose quiz categories, generate quizzes using AI, track performance, and compete through leaderboards while enjoying a smooth app-like experience across desktop, tablet, and mobile devices.
